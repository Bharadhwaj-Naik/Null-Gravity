 🚀 Null Gravity
 
Null Gravity is a fast-paced arcade survival game built using pure HTML, CSS, and JavaScript.
Tap or press space to thrust upward and dodge incoming obstacles.

🎮 Gameplay
* Control a neon square ship
* Avoid magenta obstacle columns
* Gain points by passing obstacles
* Build combos for consecutive passes
* Difficulty increases as your score grows

⚡ Features
* 🎨 Neon cyberpunk UI design
* 🌌 Animated starfield background
* 💥 Particle effects for interactions
* 🔊 Sound effects using Web Audio API
* 📈 Dynamic difficulty scaling
* 🏆 High score tracking (localStorage)
* 📱 Mobile + desktop responsive controls

🕹️ Controls
| Action    | Key / Input     |
| --------- | --------------- |
| Thrust Up | `SPACE` / Tap   |
| Pause     | `P` / `ESC`     |
| Restart   | `R` (Game Over) |

🛠️ Tech Stack
* HTML5 Canvas
* CSS3 (Neon UI styling)
* Vanilla JavaScript
* Web Audio API

 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Bharadhwaj-Naik/null-gravity.git
   ```
2. Open the folder:

   ```bash
   cd null-gravity
   ```

3. Run the game:

   * Open `index.html` in your browser
